This is my final project in Android Studio. (Lior Furman)

The theme I chose is Board Game Tracker.
Every weekend, my friends and I get together to play board games, and I thought it would be a great idea for a project.
This app allows us to track our wins in each game. We can either see who has the most wins in each game / totally or check the history of wins,
where we can see who won at what date in which game.
